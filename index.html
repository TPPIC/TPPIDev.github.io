What the heck is this?

Test Pack Please Ignore (TPPI) is a Modpack for Minecraft.

It features a vast amount of mods, while still maintaining a balanced, stable feel.

We pride ourselves on making sure the pack feels like everything works together. Less time fighting over making mods work together means moer time playing and enjoying the game. We have gone to great lengths to improve stability and coherence, which is needed with so many mods in the pack.

Well, how can I start?

TPPI is currently available on two modpack platforms: Feed The Beast and Technic

FTB

You can find the pack under the 3rd Party Packs tab in the FTB Launcher. Simply hit launch and you are ready to go!

Technic

TPPI is the latest main modpack for the Technic platform. This means that it is available on the Technic Launcher without any special URLs or codes needed! Simply download the Technic Launcher and TPPI will be in the list of modpacks on the left. Note: You will likely need to increase the memory allocation to ATLEAST 2.5GB if using the Technic Launcher.

Do note that the pack currently needs java 7 (not 8 and certainly not 6) to run at all. Please make sure you have the proper version installed.

Enjoy.
